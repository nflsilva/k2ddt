## About The Project

A collection of very simple tools to build 2D games.

### Key features

```
-Rendering
	- Simple shapes (Circle, Donut, Quad)
	- Particles
	- Sprites
	- Sprites made from different 'sub-sprites' (MultiSprites)
	- Sprite animation
	- Sprite atlas processing
	- Batch rendering
	- Text rendering
	
- Physics
    - Collision detection
    - Verlet integration
	
-Input
	- Mouse clicks
	- Keyboard
```
